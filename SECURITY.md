# Security Policy

## Supported Versions

`netbar` is early-stage software. Security fixes will be applied to the latest version on the default branch.

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory for this repository.

Do not open a public issue for vulnerabilities that could affect users.

## Scope

Security reports should focus on issues such as:

- Unsafe file writes.
- Command execution vulnerabilities.
- Sensitive data exposure.
- Network behavior that could surprise users.
